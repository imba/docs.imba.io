# Arrays And Objects

## Creating Arrays

```ruby
['Imba', 'Svelte', 'ReactFiber']
Array('html', 'css', 'js')
```

## Accessing Arrays

```ruby
let names = ['Imba', 'Svelte', 'React']
console.log names[0]
```

## Mutate Arrays

```ruby
# Mutate Array
name[3] = 'Angular'
```

## Array Value Types

```ruby
var details = ['Imba', 2.0, 2020, 'language', true, {creator:"Sindre"}]
```

## Array methods

You can use all of the native JavaScript Array Methods

```ruby
names.indexOf('Imba')
names.pop()
names.push('Vue')
names.unshift('Javascript')
names.pop()
names.shift()
```

## Array Properties

Can use all JavaScript Array Properties

```ruby
names.length # same as Javascript
```
